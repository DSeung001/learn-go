# Markdown Editor 1

Markdown is a lightweight markup language with plain text formatting syntax.

## Features

- Tables
- Fenced code blocks
- Drag and drop markdown files
- Even More



## Code Highlighting

```javascript 
const CHANCE = Math.random() > 0.5 ? true : false; 
if (CHANCE === true) { 
console.log('It`s true'); 
} 
if (CHANCE === false) { 
console.log('It's false'); 
} 
```
